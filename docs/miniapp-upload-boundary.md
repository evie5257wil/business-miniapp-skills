# 小程序上传边界说明

本仓库当前只上传 Codex 工作流脚本和脱敏说明，不直接上传单位业务小程序原始压缩包。

## 暂不上传的内容

- 小程序正式压缩包
- 人员账号台账
- 车辆台账
- PDF 原始说明文件
- 任何包含单位名称、人员姓名、手机号、车牌号、内部流程细节的文件

## 后续如需上传小程序代码

建议先建立单独的脱敏版本：

- 替换单位名称为通用名称
- 替换人员、车辆、账号、电话、车牌等数据为示例数据
- 移除 AppID、云环境 ID、密钥、接口地址
- 去掉真实业务流程中的敏感节点
- 重新生成 README、更新记录、使用说明

## 当前状态

- 已上传：通用脚本与发布留痕工具
- 未上传：业务小程序原始文件

这个边界用于保证即使仓库将来从私有转公开，也不会直接暴露单位或个人敏感信息。
